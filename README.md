---

## 🧩 Architecture Decisions

* **Component-Based Design:**
  Each UI part is split into reusable components like SearchBar, ProductCard, etc.

* **State Management:**
  Managed using React Hooks (`useState`).

* **API Handling:**
  Data fetched using `fetch()` inside `useEffect`.

* **Separation of Concerns:**
  UI and logic are separated into different components for better readability and scalability.
